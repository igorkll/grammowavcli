# grammowavcli
## PRINT YOUR MUSIC
![preview](https://raw.githubusercontent.com/igorkll/grammowavcli/refs/heads/main/preview.png)  
converts a music file into a 3D model of a record for printing on a 3D printer and listening on a gramophone  
please note that model generation can take up to several hours  
printing is only possible on a photopolymer 3D printer, as FDM does not have enough quality  
it is not recommended to play printed records on an electric turntable, as this may damage the pickup  
the default program parameters have been calculated for printing 78 rpm records and further listening on the gramophone
